# grid
自适应网格组件。

## Usage

```html
<mx-grid>
    girdItem
</mx-grid>
```

## API 

### Props

| Attribute | Description              | Type   | Default |
| --------- | ------------------------ | ------ | ------- |
| cols      | 网格列数                 | Number | 2       |
| hMargin   | 网格上下padding，单位px  | Number | 8       |
| vMargin   | 网格左右padding，单位px  | Number | 8       |
| itemSpace | 网格元素左右间隔，单位px | Number | 8       |
| lineSpace | 网格元素上下间隔，单位px | Number | 8       |
